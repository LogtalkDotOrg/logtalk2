Welcome to the Logtalk for Mac OS X Installer

Logtalk is an open source object-oriented extension to the Prolog programming language. Integrating logic programming with object-oriented and event-driven programming, it is compatible with most Prolog compilers. It supports both prototypes and classes. In addition, it supports component-based programming through category-based composition.

This installer guides you through the steps necessary to install Logtalk for Mac OS X. Logtalk is installed on the /opt/local/share/logtalk directory. After running this installer, follow the instructions on the /opt/local/share/logtalk/INSTALL.txt file regarding copying the Logtalk user-modifiable files to your home directory. The /opt/local/share/logtalk/CUSTOMIZE.txt describes how to customize Logtalk for your personal use. Links to the Logtalk – Prolog integration scripts are created on the /opt/local/bin directory, which must be on your system path. The integration scripts are described on the /opt/local/share/logtalk/scripts/NOTES.txt file.
