sender/1

Description

sender(Sender)

Returns the object that has sent the message under processing.

Template and modes

sender(-object)

Errors

(none)

Examples

test :-
    sender(Sender),
    write('executing a method to answer a message sent by '),
    writeq(Sender), nl.
