\+/1

Description

\+ Goal

Not-provable meta-predicate. True iff call(Goal) is false. This built-in meta-predicate cannot be used as a message to an object.

Template and modes

\+ +callable

Errors

Goal is a variable:
instantiation_error
Goal is neither a variable nor a callable term:
type_error(callable, Goal)

Examples

Not-provable goal in this:
\+ Goal
Not-provable goal in self:
\+ ::Goal
Not-provable goal in an explicit object:
\+ Object::Goal
