after/3

Description

after(Object, Message, Sender)

This is a pre-declared but user-defined public method for handling after events.

Template and modes

after(?object, ?term, ?object)

Errors

(none)

Examples

after(Object, Message, Sender) :-
    writeq(Object), write('::'), writeq(Message),
    write(' from '), writeq(Sender), nl.
