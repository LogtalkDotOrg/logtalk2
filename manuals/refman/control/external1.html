contents > reference manual > control constructs Logtalk control construct: {}/1

{}/1

Description

{Goal}

Calls external Prolog code. Can be used to bypass the Logtalk preprocessor/compiler.

Template and modes

{+callable}

Errors

(none)

Examples

N1/D1 < N2/D2 :-
    {N1*D2 < N2*D1}.
