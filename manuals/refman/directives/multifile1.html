multifile/1

Description

multifile(Functor/Arity)
multifile(Functor1/Arity1, Functor2/Arity2, ...)
multifile([Functor1/Arity1, Functor2/Arity2, ...])

Declares multifile predicates. This directive can only be used with back-end Prolog compilers fully supporting multifile predicates.

Template and modes

multifile(+predicate_indicator_term)

Examples

:- multifile(table/3).
