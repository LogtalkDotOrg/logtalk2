contents > reference manual > directives

if/1

Description

if(Goal)

Starts conditional compilation. The code following the directive is compiled if Goal is true. The goal is automatically term expanded.

Template and modes

if(+callable)

Examples

:- if(current_prolog_flag(double_quotes, atom)).
