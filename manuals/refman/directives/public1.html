public/1

Description

public(Predicate)
public(Predicate1, Predicate2, ...)
public([Predicate1, Predicate2, ...])

Declares public predicates. A public predicate can be called from any object.

Template and modes

public(+predicate_indicator_term)

Examples

:- public(ancestor/1).

:- public(instance/1, instances/1).

:- public([leaf/1, leaves/1]).
