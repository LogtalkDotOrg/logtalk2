alias/2

Description

alias(Object, Alias)

Experimental directive for declaring object aliases. An object alias is an alternative name for an object that may be used to abbreviate long object names or to make it easier to switch between objects providing alternative implementations for the same protocol (e.g. by using conditional compilation directives to select between alias directives). This directive may be used in objects and categories.

Object aliases are replaced at compile-time in Alias::Message goals (as long as Alias is instantiated and the goals do not occur as arguments for user-defined meta-predicates or for non-standard Prolog meta-predicates not declared on the config files).

Template and modes

alias(@object_identifier, @object_identifier)

Examples

:- alias(red_black_tree, rbt).
