retractall/1

Description

retractall(Head)

Logtalk adds this built-in predicate, with the usual definition, to a Prolog compiler if it is not already defined.

Template and modes

retractall(+callable)

Errors

Head is not a callable term:
type_error(callable, Head)

Examples

| ?- retractall(foo(_)).
